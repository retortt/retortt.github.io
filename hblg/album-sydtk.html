四月的天空


清华89级同学创作并演唱，在母校108岁生日之际，重聚校园，绽放一样的青春和笑容。

《四月的天空》

我期待再与你相逢
我期待再握你的手
我期待拥你在怀中
我期待把你留住
我期待每一次感动
我期待四月的天空
我期待在我生命中
你和我一起与共

   

我替你送晚秋去延安 / 我替你陪老板吃便饭 / 等我回天津摆佛龛 / 我和你一起爱左蓝 / 我也想从重庆走延安 / 我也想抱着雨农撞岱山 / 我也想重回海河天津站 / 我也想梦中念左蓝 / 你们编织在华北的浪漫 / 全刻在小卧室的天花板 / 你那峨眉峰埋葬在对岸 / 渤海深处写满了不甘
